# archive/ — retired skills (provenance only)

Historical records of retired skills, preserved for the dissertation methods chapter and to explain
legacy library state. **Not packaged** into releases (the build only ships `skills/`), **not installed**,
**DO NOT EXECUTE**. Any executable scripts and embedded credentials have been removed; see each
`SKILL.md`'s "What was removed" section.

- `zotero-slr-dedup-archived/` — the supersede-based cross-source dedup workflow, retired 2026-07-16 in
  favour of `zotero-merge-prep`. Its `zotero_common.py` (deleted) had held hard-coded Zotero keys, since
  **revoked**.
